This section focuses on the public API that should remain stable across framework changes. Internal helpers are intentionally omitted unless they matter for plugins, tooling, or debugging.

Module map

QuDB

Local data core, mount routing, data/blob/delivery listeners, tombstones, and blob storage.

QuNet

Transport runtime, endpoints, connection state, and packet routing.

QuSync

Replica subscriptions, pulls, live sync, queue-backed blob upload/download, and relay-facing behaviour.

UI binding

Native qu-* attributes, qu-bind, qu-list, and native template rendering.

Blobs and delivery

Blob metadata, local and relay progress, delivery tracker states, and listener scopes.

Relay and replicas

Relay peer runtime, replica restore, local relay transport, and feature-lab wiring.
