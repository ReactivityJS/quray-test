Relay and replicas

A relay is treated as a peer-shaped transport endpoint. It can optionally persist replica data so disconnected or newly connected peers can restore state later.

Key pieces

createRelayPeer()

Node-side relay runtime used by automated tests and local relay setups.

NodeRelayTransport()

Node-only in-process transport adapter that connects a normal peer transport to the relay peer.

Browser demo relay

The interactive browser demos use a browser-safe in-memory relay helper built on LocalRelay plus an in-memory blob replica.

Local browser feature lab

The browser feature lab uses a browser-safe in-memory relay helper. Node-only relay modules stay out of the browser entry, while the visible peer → relay → peer flow remains interactive in the page.
