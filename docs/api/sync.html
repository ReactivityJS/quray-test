QuSync

QuSync links QuDB and QuNet. It turns explicit subscriptions, pulls, and queued tasks into actual relay traffic without patching db.on().

Recommended usage

await sync.subscribe("@room/chat/**", { snapshot: true, live: true })
const stop = await sync.observe("@room/chat/**", (qubit) => console.log(qubit))
const rowCount = await sync.pull("@room/chat/")

Blob work inside QuSync

Blob uploads and downloads are queued tasks. Upload progress and download progress can be visualised through queue events and blob status listeners. The feature lab includes a ready-made scenario that shows local storage, relay replica storage, and peer-two download completion.
