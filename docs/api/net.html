QuNet

QuNet manages transport plugins and endpoint state. It is deliberately lower-level than QuSync and does not decide what to replicate.

Typical lifecycle

const net = QuNet()
net.use(transportInstance, "relay")
await net.connect("relay")
await net.send("relay", { payload: { type: "peer.hello", from: myPub } })
await net.disconnectAll()

What belongs here

Transport registration

Register WebSocket, HTTP, local bridge, or in-process relay transports under readable names.

Connection state

Track whether a transport is disconnected, connecting, or connected.

Packet forwarding

Send and receive framed packets without mixing in application-level sync policy.
