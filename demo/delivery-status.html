Konzept: 6-stufiger Funnel

Jede Stufe ist ein Zustand in conf/delivery/{key} — lokal persistent, nie synchronisiert.

Sende eine Nachricht um den Funnel zu sehen.
API:
// Reaktiv auf Delivery-State hören:
qr.db.on(key, (entry, meta) => {
  console.log(entry.state) // 'local', 'relay_in', …
}, { scope: 'delivery' })

// Einmalig lesen:
const d = await qr.db.delivery.get(key)
console.log(d?.state) // 'peer_recv'

// Prüfen ob mindestens 'relay_in':
const ok = await qr.db.delivery.isAtLeast(key, 'relay_in')
