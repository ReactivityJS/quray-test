Peer-Presence (Relay)

sys/peers/** — via peer.hello / peer.bye. Reaktiv — kein manuelles Refresh.

Keine Peers online.
Simuliere Presence-Events:

Konzept: Sichtbarkeits-Kontrolle

Öffentlich (unverschlüsselt)
~pub/status → "online"
Jeder Peer mit Relay-Zugang kann den Status lesen.
Privat (self-encrypted)
~pub/status → {ct:…, by:alice.pub}
Nur Alice kann lesen. Anderen Peers: sehen nur 🔒 enc.
Selektiv (multi-recipient)
~pub/status → {ct:…, keys:{bobPub:…}}
Alice + Bob können lesen. Envelope-Verschlüsselung.
Unsichtbar
Kein peer.hello senden + Status-QuBit löschen.
Erscheint nicht in sys/peers/.
