    Manual verification checklist

    1. The inline counter inside the sentence should update without reloading the page.
    2. The denied write should produce a visible failure log and must not appear under Bob's latest room entry.
    3. The tombstone delete should make db.get() return null while the tombstone stays visible with includeDeleted: true.
    4. Adding, editing, and deleting tasks through Alice should update the Alice list and Bob mirrored count.
    5. The media blob sync should show progress bars moving and the replicated media should preview on Bob.
